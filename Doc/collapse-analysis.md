# Collapse Analysis — Pattern Recognition

A collapse is not failure — it’s an attempt at protection.

## Common Triggers
- Gaslighting  
- Boundary violations  
- Invalidation  
- Identity distortion  
- Fear of abandonment  

## Collapse Types
1. **Backward Pull** — dissociation  
2. **Forward Surge** — boundary activation  
3. **Float/Distance** — overwhelm  
4. **Freeze Drop** — immobilization  

## Tracking a Collapse
Each includes:
- Signal  
- Trigger  
- Meaning  
- Repair  
- Integration arc
