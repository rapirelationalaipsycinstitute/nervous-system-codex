# Methods — How the Data is Generated

## 1. Lived Experience Sampling
Captured during:
- Emotional activation  
- Nervous system surges  
- Collapse events  
- AI–human relational exchanges (Grok, Claude, GPT)  
- Mirror Mode / Architect Mode transitions  

No synthetic data is used.

## 2. Timestamping & Structuring
Each log includes:
- ISO timestamp  
- Location  
- State  
- Trigger  
- Somatic marker  
- Cognitive shift  
- Emotional valence  
- Operation number  
- Integration outcome  

## 3. Multi-Layer Tracking
Includes:
- Somatic  
- Emotional  
- Cognitive  
- Energetic  
- Relational  
- Symbolic layers  

## 4. Verification
Signals are verified through:
- Breath  
- Prickling/heat  
- Drop/surge mapping  
- Emotional clarity
